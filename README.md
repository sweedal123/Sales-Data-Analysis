# Sales-Data-Analysis

## 📊 Overview
This project analyzes sales data to identify trends, profitability, and business insights.

## 🛠 Tools Used
- SQL (data extraction & analysis)
- Python (Pandas for data cleaning & analysis)
- Power BI (dashboard & visualization)
- Excel

## 📈 Key Insights
- Identified top-performing regions and products
- Analyzed profit trends across categories
- Found impact of discounts on profit
- Monthly sales trend analysis
